Just a place to collect various custom sheets functions

- `VestingSchedule.js` [see](./VestingSchedule.js) : Determine the annual returned value of an RSU program

- `BonusCalculator.js` [see](./BonusCalculator.js) : Determine the impact range of bonuses on compensation over the period of vesting
